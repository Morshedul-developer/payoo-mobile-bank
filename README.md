## 🌟 About The Project

**Payoo** is a mobile-first **digital banking web application** that simulates a real-world mobile banking experience right in the browser — no app store needed. Users log in with their mobile number and a 4-digit PIN, and land on a clean dashboard where they can manage all their financial activities from one place.

The app supports **6 core banking operations** — Add Money, Cash Out, Send Money, Get Bonus, Pay Bill, and Transaction History — all accessible through a smooth, intuitive interface. Built with a mobile-first layout (`max-w-md`) to perfectly mimic a real smartphone banking app, Payoo delivers a genuine banking feel using pure HTML and JavaScript without any heavy framework.

> 📱 Login → Choose an action → Manage your money — all from one beautiful mobile interface.

## ✨ Features

### 🔐 1. Secure Login System
Users authenticate using their **11-digit mobile number** and a **4-digit PIN**, just like a real mobile banking app. After successful login, they are redirected to the main dashboard. The logout button on the dashboard safely returns the user to the login page — keeping the session flow realistic and secure.

### 💸 2. Complete Banking Operations in One Dashboard
The main dashboard offers **6 fully functional banking actions** — all in one screen:

| Action | What it Does |
|---|---|
| ➕ **Add Money** | Deposit funds from DBBL, AB Bank, or City Bank |
| 🏧 **Cash Out** | Withdraw cash via an agent number |
| 📤 **Send Money** | Transfer funds to any user account |
| 🎁 **Get Bonus** | Redeem bonus coupon codes for rewards |
| 🧾 **Pay Bill** | Pay utility bills to any biller account |
| 📋 **Transaction History** | View a complete log of all past transactions |

### 📱 3. Mobile-First UI with Real App Feel
Payoo is designed from the ground up as a **mobile banking app**, with the entire layout constrained to `max-w-md` to perfectly simulate a smartphone screen in the browser. The sticky header always shows the **available balance**, the interface is clean and card-based, and each banking section slides in and out smoothly — giving users a native app-like experience without installing anything.

## 🚀 Getting Started

No build tools or installations needed — just open the file in your browser!

```bash

git clone https://github.com/Morshedul-developer/payoo-mobile-bank.git

# 2. Navigate to the project directory
cd payoo-mobile-bank

# 3. Open index.html in your browser
# Simply double-click index.html OR use Live Server in VS Code
```

> 💡 Recommended: Use the **Live Server** extension in VS Code for the best experience.
